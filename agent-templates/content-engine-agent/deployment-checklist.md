# Content Engine Agent Deployment Checklist

## Pre-deployment
- [ ] Brand voice document provided by client
- [ ] Social platforms confirmed (LinkedIn/Instagram/TikTok)
- [ ] Posting frequency agreed
- [ ] Approval flow decided (auto-publish or human review)
- [ ] Buffer/Later account connected
- [ ] MiniMax M27 API key configured (for video content)

## Deployment
- [ ] Workflow imported to client n8n
- [ ] Schedule trigger configured (weekly Monday 6AM client timezone)
- [ ] Buffer API credentials configured
- [ ] System prompt customized with brand voice

## Testing
- [ ] Manual trigger → generates 7 content briefs
- [ ] Content written for each platform (LinkedIn, Instagram, TikTok)
- [ ] Quality review of generated content
- [ ] Activity logged in Supabase

## Post-deployment
- [ ] First week of content reviewed by client
- [ ] Adjustments to tone/style based on feedback
- [ ] Analytics tracking confirmed
